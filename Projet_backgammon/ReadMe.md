# Jeu de Backgammon 
##
Introduction :

Le programme est composé de plusieurs parties.

Côté client :
	
	•	L’affichage
	•	L’écoute du serveur.
	•	La récupération des entrées de l’utilisateur.

Côté serveur :

	•	La gestion des règles.
	•	L’écoute des clients.
